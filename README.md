# Url-Shortener
Flask web application, to save url and file with a short name.

## To run application
1. Download the zip file
2. To run flask application
  #### Install pipenv :
  ``` pip install pipenv ```
  #### After pipenv installation, tun pipenv command :
  ``` pipenv shell ``` or ``` pipenv run ```
  #### Run application :
  ``` python app.py ```
